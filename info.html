Funden is a clean, creative, unique React NextJs template that fits all kinds of crowdfunding, charity, nonprofit, NGO, donations and all other businesses, non-profit charity website, and an NGO. It has a purpose-oriented design, responsive layout, and special features like different Home pages, blog layouts, galleries, services, and pricing tables.

This template easily customizable, fully responsive and support all modern browser and device.
